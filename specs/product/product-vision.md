# 产品愿景

## 产品定义

一套面向 A-Level 国际高中场景的雅思课后 AI 教练系统，由老师主导编排、学生课后执行、家长低频感知与支持。

## 核心价值

老师主控的课后辅导放大系统 + 学生持续执行的 AI 教练系统 + 家长可理解的进展支持系统。

## 目标用户

| 角色 | 定位 |
|------|------|
| 老师 | 第一责任发起者，管理班级和训练节奏 |
| 学生 | 主要使用者，课后执行训练 |
| 家长 | 低频高价值的解释与支持端 |
| 学校管理员 | 学校级管理（老师、班级、配置） |

## MVP 核心目标

1. 老师无法高频覆盖每个学生的课后 Speaking / Writing 训练 → AI 承接
2. 学生课后练习不持续、不聚焦 → 结构化任务驱动
3. 家长看不懂进展 → 可理解的阶段说明
4. 学校个性化和可追踪性不足 → 数据化管理

## MVP 不做

选校与专业定位、文书与面试辅导、A-Level 全学科辅导、社区/课程商城、重型排课与教务系统、全量题库平台、高频家校即时沟通工具。

## 设计原则

1. Teacher-in-control — 老师掌握节奏，AI 是执行与放大器
2. 课后场景优先 — 承接课堂之后的训练
3. 先打透 Speaking / Writing — 最痛最缺高频反馈的地方
4. 学校节奏协同优先 — 围绕学校当前进度做增强
5. 家长端重解释、轻干预
6. 同一底座，双模式启动（学校版 + 家庭版，家庭版 v2）
7. 先形成三方闭环，再扩功能边界
