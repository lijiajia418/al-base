## TC-01-01: 录音中 Azure 分数区域显示等待动画

| 字段 | 值 |
|------|-----|
| Scenario | 录音进行中，Azure 列显示脉冲动画+说明文案 |
| Story | US-01-azure-waiting-state |
| Priority | 关键 |

**前置条件**：
- 页面已连接 WebSocket
- 已选择参考文本

**步骤**：
1. 点击录音按钮
2. 等待倒计时结束，开始说话
3. 观察 Azure 列

**预期结果**：
- Azure 分数区域显示脉冲动画 + 文案「录音结束后返回评测结果」
- 单词区域所有参考词灰色 + 整体呼吸灯效果
- 不显示分数数值

**清理**：停止录音
